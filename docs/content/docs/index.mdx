QueryStudio is a fast, open-source database client with a keyboard-first workflow, AI-assisted querying, and support for multiple database engines.

## Features

- **Multi-database support** - PostgreSQL, MySQL, SQLite, Redis, MongoDB
- **Keyboard-first** - Navigate and query without leaving the keyboard
- **AI query assistant** - Schema-aware chat + tool calling against your connected database
- **Custom themes** - Personalize the UI with built-in or custom color themes
- **Tab-based workflow** - Work with multiple queries and connections simultaneously
- **Query history** - Never lose a query again

## Getting Started

### Build from source

```bash
git clone https://github.com/querystudio/querystudio
cd querystudio
bun install
bun run dev
```
